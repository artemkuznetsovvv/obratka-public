# ADR-obratka-app

Веб-платформа для сбора и анализа клиентских отзывов с карт (Google Maps, 2GIS, Яндекс.Карты).

---

## Структура репозитория

### 📂 [Диаграмма компонентов](Диаграмма%20компонентов/)

Визуальная схема компонентов системы и их связей (`.drawio` + `.jpg`).

### 📂 [Диаграмма последовательностей](Диаграмма%20последовательностей%20(разовый%20анализ)/)

Sequence-диаграмма основного флоу — разовый анализ отзывов (Mermaid-исходник + `.png`).

### 📂 [ADR по компонентам системы + описание системы](ADR%20по%20компонентам%20системы%20+%20описание%20системы/)

#### [`overview.md`](ADR%20по%20компонентам%20системы%20+%20описание%20системы/overview.md) — основной файл

Главный документ с описанием компонентов системы **без глубоких деталей реализации**.
Содержит только ключевые параметры каждого компонента:

- технологический стек
- используемые БД
- способы взаимодействия между компонентами

Это отправная точка — начинайте чтение с него.

#### Отдельно стоит обратить внимание

- **[ADR-011 — Декомпозиция сервисов для MVP](ADR%20по%20компонентам%20системы%20+%20описание%20системы/details/ADR-011-mvp-service-decomposition.md)** — описывает, как и почему пришёл к конкретной сервисной декомпозиции (Modular Monolith вместо микросервисов на этапе MVP).
